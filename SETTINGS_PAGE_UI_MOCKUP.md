## Color Scheme

- **Primary**: Blue (#2563eb) for buttons, links, active states
- **Success**: Green (#16a34a) for positive actions, active status
- **Warning**: Yellow (#eab308) for warnings, alerts
- **Error**: Red (#dc2626) for errors, destructive actions
- **Gray**: Various shades for text, borders, backgrounds
- **Purple**: (#7c3aed) for super admin badges

## Interactive Elements

- **Toggle Switches**: Blue when ON, gray when OFF with sliding animation
- **Buttons**: Blue primary, gray secondary, red destructive
- **Modals**: Centered overlay with dark backdrop
- **Form Inputs**: Gray border, blue focus ring
- **Badges**: Rounded pills with background colors matching their context
- **Status Indicators**: Colored dots (green = active, blue = syncing, gray = inactive)

## Responsive Design

- **Desktop (>1024px)**: 2-column layouts, expanded navigation
- **Tablet (768-1024px)**: Single column with full-width cards
- **Mobile (<768px)**: Stacked layout, full-width everything, collapsible sections
